# 火影世界书

这个文件夹保存项目可直接读取的世界书资料。

文件说明：

- `timeline.js`：火影主时间线目录，包含远古、战国、忍界大战、木叶48年、原作第一部、疾风传、战后新时代。
- `timeline-detailed.js`：更细粒度的木叶纪年、忍界大战、村内事件、外交事件和篇章节点。
- `era-consistency.js`：当前时间线判定与年代合理性规则，防止把疾风传、原作第一部或战后结果倒灌到玩家当前年代。
- `arcs.js`：原作主要篇章目录，包含第一部、卡卡西外传、疾风传前后半与战后结构。
- `characters.js`：重要角色摘要，覆盖木叶、砂隐、雾隐、岩隐、云隐、晓、大筒木、尾兽等关键人物。
- `character-details.js`：角色性格、说话方式、动机、弱点、行为边界与木叶48年状态细化。
- `character-details-2.js`：第二批角色细节，补充尾兽、人柱力互动、晓搭档规则和时代一致性。
- `character-appearances.js`：角色外观、服饰、辨识点和场景描写辅助资料。
- `expanded-characters.js`：扩展 NPC、人柱力、尾兽、家族成员、小国与日常地点。
- `locations-organizations.js`：忍村、组织、家族、能力体系资料。
- `shinobi-roster-2.js`：忍者名册扩展，补充更多角色、阵营、年龄阶段和出场边界。
- `boruto-era.js`：博人传/新时代资料；默认木叶48年主线只作为远期背景或新时代团资料。
- `world-expansion.js`：火之国地理、木叶日常地点、低阶任务、小国、经济、情报公开等级等跑团常用补充。
- `systems.js`：查克拉、性质变化、忍术等级、瞳术、封印术、医疗、幻术、傀儡术等系统资料。
- `index.js`：统一导出入口，供 `knowledge-base.js` 合并检索。

写作原则：

- 使用原创摘要，避免复制百科原文。
- 以跑团可用性优先，强调人物当前时代状态、可用支线、避免过早剧透。
- 默认游戏时代为 `木叶48年`，但玩家可以通过角色设定、存档、时间线跳跃或明确选择进入其他年代。若世界书中人物在当前时代尚未活跃，应由 AI 按当前时间线处理，不得默认玩家处于疾风传开始。
